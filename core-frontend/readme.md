# EarthLotto

## addPlatform

```bash

  # platform - android, ios]

  // 프로젝트 세팅
  npm i

  // 프로젝트 빌드
  npm run build-<platform>

  // 프로덕션 빌드
  npm run build-<platform>-prod

  // 프로젝트 빌드 후 JS 변경사항만 반영
  ionic capacitor build <platform>

  // open
  ionic capacitor open <platform>

  // 바로실행
  ionic capacitor run <platform>

```
